# Искусственный интеллект и машинное обучение в современной медицине

Учебный репозиторий к пособию. Содержит минимальные шаблоны кода и демонстрационные/синтетические данные для практических заданий.

## Структура

- `code/` — Python-скрипты для выполнения заданий.
- `notebooks/` — место для Jupyter/Colab-ноутбуков; скрипты из `code/` можно перенести туда блоками.
- `data/` — учебные данные. Все файлы демонстрационные, синтетические или предназначены для учебного использования.
- `docs/` — методические пояснения, карта файлов, критерии оценки.

## Минимальные требования

Python 3.10+

```bash
pip install pandas numpy matplotlib scikit-learn
```

Для базовых заданий не требуются GPU, специализированные медицинские библиотеки и доступ к закрытым базам данных.

## Как работать студенту

1. Скачайте репозиторий.
2. Установите зависимости.
3. Откройте нужный файл из папки `code/`.
4. Проверьте, что путь к данным соответствует `data/...`.
5. Запустите код и подготовьте краткий отчёт: цель, данные, действия, результат, медицинская интерпретация, ограничения.

## Важно

Данные в репозитории не предназначены для диагностики и не являются реальными клиническими данными пациентов. Они используются только для обучения принципам анализа медицинских данных и оценки ИИ-моделей.
